# umijs-starter

这是一个 `UmiJS` + `TailwindCSS` + `TypeScript` + `ESLint` + `Prettier` 的起手式

这个工程的搭建笔记可以[在这里查看](https://welives.github.io/blog/front-end/engineering/umijs.html)

## 使用

```sh
pnpm install
```

### 开发模式

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```

### 代码规范检查

```sh
pnpm lint
```
